# Price-Discount-Calculator (Console)

Небольшой инструмент для расчёта цен по скидкам и выгоду от скидки.

[![.NET CI](https://github.com/cut9/Price-Discount-Calculator/actions/workflows/dotnet-ci-windows.yml/badge.svg)](https://github.com/cut9/Price-Discount-Calculator/actions)
[![Release](https://img.shields.io/github/v/release/cut9/Price-Discount-Calculator)](https://github.com/cut9/Price-Discount-Calculator/releases)

---

## Информация
- Платформа: кроссплатформенная консоль (.NET)
- Язык: C# .NET 8
- Тип: Консольное приложение (CLI)

---

## Быстрый старт — запустить релиз (рекомендуется для обычных пользователей)

1. Перейдите в раздел **Releases**: https://github.com/cut9/Price-Discount-Calculator/releases  
2. Скачайте архив `win-x64.rar` для вашей платформы.  
3. Распакуйте архив и дважды кликните `Price-Discount-Calculator.exe`.

---

## Быстрый старт — запустить из исходников (для разработчиков)

### Команды (PowerShell / CMD)
```powershell
git clone https://github.com/cut9/Price-Discount-Calculator.git
cd Price-Discount-Calculator
dotnet restore
dotnet run --project ./Price-Discount-Calculator/Price-Discount-Calculator.csproj
```
